# Laravel：PHP開発者のための完全なエコシステム

Laravelは、PHP開発者にとって最も生産的なWebアプリケーションフレームワークです。エレガントな構文、充実したエコシステム、そして活発なコミュニティにより、モダンなWebアプリケーションの構築が驚くほど簡単になります。

## Laravelとは

[Laravel](https://laravel.com/)は、Taylor Otwellによって2011年に開発されたオープンソースのPHPフレームワークです。「Web職人のための道具」をコンセプトに、開発者体験を最優先に設計されています。

### Laravelが選ばれる理由

**生産性の向上**：認証、ルーティング、セッション、キャッシュなど、モダンなWebアプリに必要な機能がすべて揃っています。ボイラープレートコードを書く時間を削減し、ビジネスロジックに集中できます。

**エレガントな構文**：コードが自己説明的で読みやすく、保守性が高まります。「コードは書くよりも読まれる」という哲学が貫かれています。

**完全なエコシステム**：フレームワーク本体だけでなく、デプロイ、監視、決済、検索など、あらゆる場面で使える公式パッケージが用意されています。

**活発なコミュニティ**：世界中の開発者が利用しており、Laracast（動画学習）、Laravel News、フォーラムなど、学習リソースが豊富です。

## 主要機能

Laravelは、現代のWebアプリケーション開発に必要な機能を標準で提供します。

### 認証（Authentication）

ユーザー認証は、わずか数行で実装できます。

```php
Route::get('/profile', ProfileController::class)
    ->middleware('auth');
```

認証済みユーザーへのアクセスは、Authファサードを使用します。

```php
use Illuminate\Support\Facades\Auth;

$user = Auth::user();
```

Laravelのスターターキット（後述）を使えば、登録、ログイン、パスワードリセット、メール確認、プロフィール設定など、完全な認証フローが最初から含まれます。

### 認可（Authorization）

認証後のアクセス制御は、ポリシー（Policy）で定義します。

```php
public function update(User $user, Post $post): Response
{
    return $user->id === $post->user_id
                ? Response::allow()
                : Response::deny('You do not own this post.');
}
```

Gateファサードで簡単にチェックできます。

```php
Gate::authorize('update', $post);
```

ポリシーベースの認可により、複雑な権限ロジックも整理された状態を保てます。

### Eloquent ORM

Laravelの心臓部とも言えるEloquent ORMは、データベース操作を直感的にします。

モデル定義は、シンプルなクラスで完結します。

```php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class Flight extends Model
{
    // ...
}
```

データの取得は、メソッドチェーンで記述します。

```php
use App\Models\Flight;

foreach (Flight::all() as $flight) {
    echo $flight->name;
}
```

複雑なクエリも、可読性を損なわず記述できます。

```php
$flights = Flight::where('active', 1)
    ->orderBy('name')
    ->take(10)
    ->get();
```

リレーションシップも宣言的に定義でき、N+1問題の回避も自動で行われます。

```php
// 1対多のリレーション
public function posts()
{
    return $this->hasMany(Post::class);
}

// Eager Loading で N+1 を回避
$users = User::with('posts')->get();
```

### マイグレーション（Migrations）

データベーススキーマをコードで管理できるため、チーム開発で環境を統一しやすくなります。

```bash
php artisan make:migration create_posts_table --create=posts
```

スキーマ定義は、流暢なAPIで記述します。

```php
class CreatePostsTable extends Migration
{
    public function up()
    {
        Schema::create('posts', function (Blueprint $table) {
            $table->id();
            $table->string('title');
            $table->text('content');
            $table->foreignId('user_id')->constrained()->onDelete('cascade');
            $table->timestamps();
        });
    }
}
```

実行は、Artisanコマンド一つで完了します。

```bash
php artisan migrate
```

### バリデーション（Validation）

入力検証は、シンプルかつ強力です。

```php
$validated = $request->validate([
    'title' => 'required|max:255',
    'email' => 'required|email|unique:users',
    'password' => 'required|min:8|confirmed',
]);
```

カスタムルールの作成も簡単です。

```php
$request->validate([
    'email' => ['required', 'email', new CustomRule()],
]);
```

エラーメッセージは自動で生成され、多言語対応も標準でサポートされています。

### ルーティング（Routing）

ルート定義は、表現力豊かで簡潔です。

```php
Route::get('/users', [UserController::class, 'index']);
Route::post('/users', [UserController::class, 'store']);
Route::get('/users/{id}', [UserController::class, 'show']);
```

リソースルートを使えば、RESTful APIのルートを一括定義できます。

```php
Route::resource('posts', PostController::class);
```

これだけで、index、create、store、show、edit、update、destroyの7つのルートが自動生成されます。

## エコシステム

Laravelの真の強みは、フレームワーク本体だけでなく、豊富な公式パッケージにあります。

### 検索（Scout）

Eloquentモデルに全文検索機能を追加できます。

```php
use Laravel\Scout\Searchable;

class Post extends Model
{
    use Searchable;
}

// 検索実行
$posts = Post::search('laravel')->get();
```

Algolia、Meilisearch、Typesense、データベースドライバーなど、複数のエンジンをサポートしています。

### 高性能アプリサーバー（Octane）

LaravelアプリケーションをRoadRunnerやFrankenPHPで動作させ、劇的なパフォーマンス向上を実現します。従来のPHP-FPMと比較して、数倍から数十倍の高速化が可能です。

### WebSocket（Reverb）

リアルタイム通信を実現する、Laravel製のWebSocketサーバーです。

```php
// イベントをブロードキャスト
broadcast(new OrderShipped($order));
```

クライアント側では、Laravel Echoで簡単にリッスンできます。

```javascript
Echo.channel('orders').listen('OrderShipped', (e) => {
  console.log(e.order)
})
```

### 機能フラグ（Pennant）

A/Bテストや段階的なロールアウトに使える機能フラグ管理システムです。

```php
use Laravel\Pennant\Feature;

if (Feature::active('new-ui')) {
    // 新UIを表示
}
```

### 決済（Cashier）

StripeやPaddleとの統合を簡単にします。サブスクリプション、単発決済、請求書の管理が数行で実装できます。

```php
$user->newSubscription('default', 'price_premium')
    ->create($paymentMethod);
```

### ソーシャル認証（Socialite）

OAuth認証を実装するためのパッケージです。Google、Facebook、Twitter、GitHubなど、主要なプロバイダーに対応しています。

```php
return Socialite::driver('github')->redirect();
```

### API認証（Sanctum）

SPA（Single Page Application）やモバイルアプリ向けのトークンベース認証を提供します。

```php
Route::get('/api/user', function (Request $request) {
    return $request->user();
})->middleware('auth:sanctum');
```

### ローカル開発（Sail）

Dockerベースの開発環境を、一つのコマンドで起動できます。

```bash
./vendor/bin/sail up
```

PHP、MySQL、Redis、Mailpit、Seleniumなど、必要なサービスがすべて含まれます。

### コードスタイラー（Pint）

PHPコードを自動で整形します。Laravelのコーディング規約に準拠した美しいコードを保ちます。

```bash
./vendor/bin/pint
```

### キュー監視（Horizon）

Redisキューの監視ダッシュボードです。ジョブの実行状況、失敗ログ、メトリクスを可視化します。

### ブラウザテスト（Dusk）

エンドツーエンドのブラウザテストを記述できます。

```php
$browser->visit('/login')
        ->type('email', 'user@example.com')
        ->type('password', 'password')
        ->press('Login')
        ->assertPathIs('/dashboard');
```

### デバッグ（Telescope）

ローカル環境でのデバッグを支援します。リクエスト、例外、ログ、クエリ、キュー、メールなどを記録し、美しいUIで確認できます。

### パフォーマンス監視（Pulse）

本番環境でのパフォーマンスインサイトを提供します。遅いエンドポイントやジョブを特定し、ボトルネックを解消できます。

## スターターキット

Laravelは、フロントエンドフレームワークと組み合わせたスターターキットを提供しています。すべて無料で利用でき、認証機能が最初から含まれます。

### React Starter Kit

Laravel + Inertia + React + Tailwind CSSの構成です。

```bash
composer create-project laravel/laravel example-app
cd example-app
php artisan breeze:install react
```

シングルページアプリケーション（SPA）のような体験を、APIを構築せずに実現できます。

### Vue Starter Kit

Laravel + Inertia + Vue + Tailwind CSSの構成です。

```bash
php artisan breeze:install vue
```

Vueの柔軟性とLaravelのバックエンドの強みを組み合わせられます。

### Livewire Starter Kit

Laravel + Livewire + Tailwind CSSの構成です。

```bash
php artisan breeze:install livewire
```

JavaScriptのフレームワークを使わず、PHPだけで動的なUIを構築できます。

すべてのスターターキットに、以下が含まれます：

- 登録
- ログイン
- パスワードリセット
- メール確認
- プロフィール設定
- ダッシュボード
- ライト/ダークモード切り替え
- オプションでWorkOS AuthKitサポート

## フロントエンド統合

Laravelは、現代的なフロントエンド開発パターンに対応しています。

### Inertia：モダンモノリス

Inertiaは、LaravelとReact/Vue/Svelteをシームレスに統合します。APIを構築することなく、モダンなSPAを実現できます。

```php
use Inertia\Inertia;

class UserController extends Controller
{
    public function index()
    {
        $users = User::active()
            ->orderByName()
            ->get(['id', 'name', 'email']);

        return Inertia::render('Users', [
            'users' => $users,
        ]);
    }
}
```

React側では、propsとして受け取ります。

```jsx
export default ({ users }) => {
  return (
    <div>
      <h1>Users</h1>
      <ul>
        {users.map((user) => (
          <li key={user.id}>{user.name}</li>
        ))}
      </ul>
    </div>
  )
}
```

ルーティングやデータ転送は自動で処理され、2つのルート定義を保守する必要がありません。

### Livewire：スーパーパワードBlade

Livewireは、Bladeテンプレートに動的なリアクティブ機能を追加します。JavaScriptを書かずに、モダンなUIを構築できます。

```php
class Counter extends Component
{
    public $count = 1;

    public function increment()
    {
        $this->count++;
    }

    public function decrement()
    {
        $this->count--;
    }

    public function render()
    {
        return view('livewire.counter');
    }
}
```

テンプレートは、シンプルです。

```blade
<div>
    <h1>{{ $count }}</h1>

    <button wire:click="increment">+</button>

    <button wire:click="decrement">-</button>
</div>
```

クリックイベントで、サーバー側のメソッドが呼ばれ、UIが自動で更新されます。

### SPA・モバイルアプリ

LaravelをAPIバックエンドとして使用し、Vue.js、React、React Native、Flutterなどと組み合わせることもできます。

Sanctumを使えば、トークンベースの認証が簡単に実装できます。

```php
Route::get('/api/user', function (Request $request) {
    return $request->user();
})->middleware('auth:sanctum');
```

RESTful APIやGraphQLにも対応しており、柔軟なアーキテクチャを選択できます。

## デプロイと監視

Laravelは、デプロイと運用も簡単にします。

### Laravel Cloud：フルマネージドプラットフォーム

新しくリリースされたLaravel Cloudは、Laravelアプリケーション専用のフルマネージドプラットフォームです。

- **従量課金制**：使った分だけ支払い
- **自動スケーリング**：トラフィックに応じて自動拡張
- **ゼロダウンタイムデプロイ**：本番環境への影響なし
- **グローバルCDN**：世界中で高速配信

開発者は、コードを書くことだけに集中できます。

### Forge：サーバー管理

VPSサーバーでの運用を簡単にするツールです。

- **即座のプロビジョニング**：DigitalOcean、AWS、Vultr、Hetznerなどに対応
- **ゼロダウンタイムデプロイ**：本番環境への影響を最小化
- **SSL証明書の自動更新**：Let's Encryptと統合
- **スケジューラー管理**：Cronジョブを簡単に設定

月額$12から利用でき、無制限のアプリケーションをデプロイできます。

### Nightwatch：監視とインサイト

Laravel専用の監視ツールです。

- **パフォーマンス監視**：遅いエンドポイントやジョブを特定
- **ログ管理**：構造化ログの検索と分析
- **エラー追跡**：例外の自動通知とスタックトレース
- **無料で始められる**：基本プランは無料

本番環境で何が起きているかを正確に把握できます。

### Pulse：パフォーマンスインサイト

オープンソースのパフォーマンス監視パッケージです。無料で利用でき、以下の情報を可視化します：

- リクエストのレイテンシ（p95、p99）
- 遅いクエリ
- 遅いジョブ
- アクティブユーザー数
- CPU・メモリ使用率

### Telescope：ローカルデバッグ

開発環境でのデバッグを劇的に改善します。以下の情報を記録・可視化します：

- リクエスト
- 例外
- ログ
- データベースクエリ
- キューイングされたジョブ
- メール
- 通知
- キャッシュ操作
- スケジューラー実行
