# Laravel入門完全ガイド：PHPフレームワークの王者を初心者向けに解説

[Laravel](https://laravel.com/)は、PHPで最も人気のあるWebアプリケーションフレームワークです。エレガントな構文と豊富な機能により、開発者は効率的にWebアプリケーションを構築できます。初心者にも分かりやすいLaravelの基本から実践的な使い方まで詳しく解説します。

## Laravelとは何か

Laravelは、Taylor Otwellによって2011年に開発されたPHPフレームワークです。以下の特徴があります：

- **エレガントな構文**: 読みやすく、書きやすいコード
- **豊富な機能**: 認証、ルーティング、データベース操作などが標準搭載
- **MVCアーキテクチャ**: 保守性の高いコード構造
- **Eloquent ORM**: 直感的なデータベース操作
- **強力なエコシステム**: 多数の公式パッケージとサードパーティライブラリ
- **活発なコミュニティ**: 世界中の開発者による継続的な改善

## Laravelのインストール

### 前提条件

- PHP 8.1以上
- Composer（PHPの依存関係管理ツール）
- データベース（MySQL、PostgreSQL、SQLite等）

### インストール手順

```bash
# Composerを使用したLaravelのインストール
composer create-project laravel/laravel my-app

# プロジェクトディレクトリに移動
cd my-app

# 開発サーバーの起動
php artisan serve
```

ブラウザで `http://localhost:8000` にアクセスすると、Laravelのウェルカムページが表示されます。

## Eloquent ORMの基本

### モデルの作成

```php
// app/Models/User.php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class User extends Model
{
    // テーブル名（省略可能、デフォルトはモデル名の複数形）
    protected $table = 'users';

    // 主キー（省略可能、デフォルトは'id'）
    protected $primaryKey = 'id';

    // タイムスタンプの自動管理
    public $timestamps = true;

    // 一括代入可能な属性
    protected $fillable = [
        'name',
        'email',
        'password'
    ];

    // 一括代入から除外する属性
    protected $guarded = [
        'id',
        'created_at',
        'updated_at'
    ];

    // 属性のキャスト
    protected $casts = [
        'email_verified_at' => 'datetime',
        'is_active' => 'boolean'
    ];
}
```

### 基本的なデータベース操作

```php
// ユーザーの作成
$user = User::create([
    'name' => '田中太郎',
    'email' => 'tanaka@example.com',
    'password' => bcrypt('password123')
]);

// 全ユーザーの取得
$users = User::all();

// 条件付きクエリ
$activeUsers = User::where('is_active', true)->get();

// 単一レコードの取得
$user = User::find(1);
$user = User::where('email', 'tanaka@example.com')->first();

// ユーザーの更新
$user = User::find(1);
$user->name = '佐藤花子';
$user->save();

// または
User::where('id', 1)->update(['name' => '佐藤花子']);

// ユーザーの削除
$user = User::find(1);
$user->delete();

// または
User::destroy(1);
```

### リレーションシップの定義

```php
// app/Models/User.php

{
    // ユーザーは複数の投稿を持つ
    public function posts()

    // ユーザーは複数のコメントを持つ

    {
        return $this->hasMany(Comment::class);
    }

// app/Models/Post.php
class Post extends Model
{
    // 投稿は1人のユーザーに属する
    public function user()
    {
        return $this->belongsTo(User::class);
    }

    // 投稿は複数のコメントを持つ
    public function comments()
    {
        return $this->hasMany(Comment::class);
    }

    // 投稿は複数のタグを持つ（多対多）
    public function tags()
    {
        return $this->belongsToMany(Tag::class);
    }
}
```

### 認証されたユーザーの取得

```php
// app/Http/Controllers/UserController.php
use Illuminate\Support\Facades\Auth;

class UserController extends Controller
{
    public function profile()
    {
        $user = Auth::user();
        return view('profile', compact('user'));
    }

    public function updateProfile(Request $request)
    {
        $user = Auth::user();

        $request->validate([
            'name' => 'required|string|max:255',
            'email' => 'required|email|unique:users,email,' . $user->id,
        ]);

        $user->update($request->only(['name', 'email']));

        return redirect()->route('profile')->with('success', 'プロフィールを更新しました');
    }
}
```

## バリデーション

### フォームリクエストクラス

```php
// app/Http/Requests/StoreUserRequest.php
<?php

namespace App\Http\Requests;

use Illuminate\Foundation\Http\FormRequest;

class StoreUserRequest extends FormRequest
{
    public function authorize()
    {
        return true; // 認可ロジック
    }

    public function rules()
    {
        return [
            'name' => 'required|string|max:255',
            'email' => 'required|email|unique:users,email',
            'password' => 'required|string|min:8|confirmed',
            'age' => 'required|integer|min:18|max:120',
        ];
    }

    public function messages()
    {
        return [
            'name.required' => '名前は必須です',
            'email.required' => 'メールアドレスは必須です',
            'email.email' => '有効なメールアドレスを入力してください',
            'email.unique' => 'このメールアドレスは既に使用されています',
            'password.required' => 'パスワードは必須です',
            'password.min' => 'パスワードは8文字以上で入力してください',
            'password.confirmed' => 'パスワードの確認が一致しません',
        ];
    }
}
```

### コントローラーでのバリデーション

```php
// app/Http/Controllers/UserController.php
use App\Http\Requests\StoreUserRequest;

class UserController extends Controller

    }
}
```

```php
use Illuminate\Database\Migrations\Migration;
use Illuminate\Database\Schema\Blueprint;

            $table->id();
            $table->string('name');
            $table->string('email')->unique();
            $table->timestamp('email_verified_at')->nullable();
            $table->string('password');
            $table->boolean('is_active')->default(true);
            $table->rememberToken();
            $table->timestamps();
        });
    }

    public function down()
    {
        Schema::dropIfExists('users');
    }
}
```

## 実践的なアプリケーション例

### ブログアプリケーションの作成

```php
// app/Models/Post.php
<?php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class Post extends Model
{
    protected $fillable = [
        'title',
        'content',
        'user_id',
        'published_at'
    ];

    protected $casts = [
        'published_at' => 'datetime'
    ];

    public function user()
    {
        return $this->belongsTo(User::class);
    }

    public function comments()
    {
        return $this->hasMany(Comment::class);
    }

    public function scopePublished($query)
    {
        return $query->whereNotNull('published_at');
    }
}
```

```php
// app/Http/Controllers/PostController.php
<?php

namespace App\Http\Controllers;

use App\Models\Post;
use Illuminate\Http\Request;

class PostController extends Controller
{
    public function index()
    {
        $posts = Post::with('user')
            ->published()
            ->latest('published_at')
            ->paginate(10);

        return view('posts.index', compact('posts'));
    }

    public function show(Post $post)
    {
        $post->load(['user', 'comments.user']);
        return view('posts.show', compact('post'));
    }

    public function create()
    {
        return view('posts.create');
    }

    public function store(Request $request)
    {
        $request->validate([
            'title' => 'required|string|max:255',
            'content' => 'required|string',
        ]);

        $post = Post::create([
            'title' => $request->title,
            'content' => $request->content,
            'user_id' => auth()->id(),
            'published_at' => now(),
        ]);

        return redirect()->route('posts.show', $post)
            ->with('success', '投稿を作成しました');
    }
}
```
