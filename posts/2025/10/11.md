# Vue.js入門完全ガイド：初心者でもわかるプログレッシブJavaScriptフレームワーク

Vue.jsは、Webユーザーインターフェース構築のための親しみやすく、パフォーマンスと汎用性の高いプログレッシブJavaScriptフレームワークです。この記事では、初心者にも分かりやすく、Vue.jsの特徴、基本的な使い方、コンポーネント、リアクティビティ、ルーティング、状態管理まで実際のコード例と共に詳しく解説します。

## Vue.jsとは

Vue.jsは、Evan Youによって開発されたJavaScriptフレームワークです。[Vue.js公式サイト](https://ja.vuejs.org/)によると、「親しみやすく、パフォーマンスと汎用性の高いフレームワーク」として設計されています。

### Vue.jsの特徴

- **親しみやすい**: 直感的なAPIとワールドクラスのドキュメント
- **高パフォーマンス**: 真にリアクティブなコンパイラで最適化されたレンダリングシステム
- **多用途**: ライブラリーとフル機能のフレームワークの間でスケールする
- **段階的採用**: 既存のプロジェクトに部分的に導入可能

## 基本的な使い方

### インストール

```bash
# CDN経由で使用
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

# npm経由でインストール
npm install vue@next

# Vue CLIを使用
npm install -g @vue/cli
vue create my-project
```

## リアクティビティ

Vue.jsの最大の特徴の一つがリアクティビティです。データが変更されると、自動的にDOMが更新されます。

### 基本的なリアクティビティ

```vue
<template>
  <div>
    <h2>リアクティビティの例</h2>
    <input v-model="message" placeholder="メッセージを入力" />
    <p>入力されたメッセージ: {{ message }}</p>

    <h3>計算プロパティ</h3>
    <p>文字数: {{ messageLength }}</p>
    <p>逆順: {{ reversedMessage }}</p>

    <h3>ウォッチャー</h3>
    <p>変更回数: {{ changeCount }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: '',
      changeCount: 0,
    }
  },
  computed: {
    messageLength() {
      return this.message.length
    },
    reversedMessage() {
      return this.message.split('').reverse().join('')
    },
  },
  watch: {
    message(newValue, oldValue) {
      this.changeCount++
      console.log(`メッセージが "${oldValue}" から "${newValue}" に変更されました`)
    },
  },
}
</script>
```

### リアクティブなオブジェクトと配列

```vue
<template>
  <div>
    <h2>リアクティブなオブジェクトと配列</h2>

    <h3>ユーザー情報</h3>
    <input v-model="user.name" placeholder="名前" />
    <input v-model="user.email" placeholder="メールアドレス" />
    <p>名前: {{ user.name }}</p>
    <p>メール: {{ user.email }}</p>

    <h3>タスクリスト</h3>
    <input v-model="newTask" @keyup.enter="addTask" placeholder="新しいタスク" />
    <ul>
      <li v-for="(task, index) in tasks" :key="index">
        {{ task }}
        <button @click="removeTask(index)">削除</button>
      </li>
    </ul>
  </div>
</template>

<script>
export default {
  data() {
    return {
      user: {
        name: '',
        email: '',
      },
      tasks: [],
      newTask: '',
    }
  },
  methods: {
    addTask() {
      if (this.newTask.trim()) {
        this.tasks.push(this.newTask.trim())
        this.newTask = ''
      }
    },
    removeTask(index) {
      this.tasks.splice(index, 1)
    },
  },
}
</script>
```

## ライフサイクルフック

Vue.jsでは、コンポーネントのライフサイクルの各段階で実行されるフックを提供しています。

```vue
<template>
  <div>
    <h2>ライフサイクルフックの例</h2>
    <p>コンポーネントが作成されてから {{ timeElapsed }} 秒経過しました</p>
    <button @click="updateData">データを更新</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      timeElapsed: 0,
      interval: null,
    }
  },
  beforeCreate() {
    console.log('beforeCreate: インスタンスが作成される前')
  },
  created() {
    console.log('created: インスタンスが作成された後')
    this.interval = setInterval(() => {
      this.timeElapsed++
    }, 1000)
  },
  beforeMount() {
    console.log('beforeMount: DOMにマウントされる前')
  },
  mounted() {
    console.log('mounted: DOMにマウントされた後')
  },
  beforeUpdate() {
    console.log('beforeUpdate: データが更新される前')
  },
  updated() {
    console.log('updated: データが更新された後')
  },
  beforeUnmount() {
    console.log('beforeUnmount: インスタンスが破棄される前')
    if (this.interval) {
      clearInterval(this.interval)
    }
  },
  unmounted() {
    console.log('unmounted: インスタンスが破棄された後')
  },
  methods: {
    updateData() {
      this.timeElapsed = 0
    },
  },
}
</script>
```

## 状態管理（Pinia）

Piniaは、Vue.jsの公式状態管理ライブラリです。

### インストールとセットアップ

```bash
npm install pinia
```

```javascript
// main.js
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'
import router from './router'

const app = createApp(App)
app.use(createPinia())
app.use(router)
app.mount('#app')
```

### ストアの作成

```javascript
// stores/user.js
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    users: [
      { id: 1, name: '田中太郎', email: 'tanaka@example.com' },
      { id: 2, name: '佐藤花子', email: 'sato@example.com' },
    ],
    currentUser: null,
  }),

  getters: {
    getUserById: (state) => (id) => {
      return state.users.find((user) => user.id === id)
    },
    userCount: (state) => state.users.length,
  },

  actions: {
    addUser(user) {
      const newUser = {
        id: Date.now(),
        ...user,
      }
      this.users.push(newUser)
    },

    updateUser(id, updates) {
      const user = this.users.find((user) => user.id === id)
      if (user) {
        Object.assign(user, updates)
      }
    },

    deleteUser(id) {
      this.users = this.users.filter((user) => user.id !== id)
    },

    setCurrentUser(user) {
      this.currentUser = user
    },
  },
})
```

## まとめ

Vue.jsは、親しみやすく、パフォーマンスと汎用性の高いプログレッシブJavaScriptフレームワークです。[Vue.js公式サイト](https://ja.vuejs.org/)で提供されている豊富なドキュメントと、直感的なAPIにより、初心者でも簡単に学習できます。

### Vue.jsの主な特徴

- **親しみやすい**: 標準的なHTML、CSS、JavaScriptをもとに構築
- **高パフォーマンス**: 真にリアクティブなコンパイラで最適化されたレンダリングシステム
- **多用途**: ライブラリーとフル機能のフレームワークの間でスケール
- **段階的採用**: 既存のプロジェクトに部分的に導入可能

### 学習のポイント

- **基本的な構文**から始める
- **リアクティビティ**の概念を理解する
- **コンポーネント**の作成と使用方法を学ぶ
- **ルーティング**と**状態管理**を実践する
- **実際のプロジェクト**で使用して経験を積む

### 推奨される学習順序

1. **基本的な構文** - テンプレート、データバインディング
2. **ディレクティブ** - v-if、v-for、v-modelなど
3. **コンポーネント** - 再利用可能なUI部品の作成
4. **ライフサイクルフック** - コンポーネントのライフサイクル
5. **Vue Router** - ルーティングの実装
6. **Pinia** - 状態管理の実装

Vue.jsは、初心者から上級者まで、様々なレベルの開発者に適したフレームワークです。まずは基本的な使い方から始めて、徐々に高度な機能を学習していきましょう。
