Vue.js入門完全ガイド：初心者でもわかるプログレッシブJavaScriptフレームワーク ​

Vue.jsは、Webユーザーインターフェース構築のための親しみやすく、パフォーマンスと汎用性の高いプログレッシブJavaScriptフレームワークです。この記事では、初心者にも分かりやすく、Vue.jsの特徴、基本的な使い方、コンポーネント、リアクティビティ、ルーティング、状態管理まで実際のコード例と共に詳しく解説します。

Vue.jsとは ​

Vue.jsは、Evan Youによって開発されたJavaScriptフレームワークです。Vue.js公式サイトによると、「親しみやすく、パフォーマンスと汎用性の高いフレームワーク」として設計されています。

Vue.jsの特徴 ​

  • 親しみやすい: 直感的なAPIとワールドクラスのドキュメント
  • 高パフォーマンス: 真にリアクティブなコンパイラで最適化されたレンダリングシステム
  • 多用途: ライブラリーとフル機能のフレームワークの間でスケールする
  • 段階的採用: 既存のプロジェクトに部分的に導入可能

基本的な使い方 ​

インストール ​

bash
# CDN経由で使用
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

# npm経由でインストール
npm install vue@next

# Vue CLIを使用
npm install -g @vue/cli
vue create my-project

リアクティビティ ​

Vue.jsの最大の特徴の一つがリアクティビティです。データが変更されると、自動的にDOMが更新されます。

基本的なリアクティビティ ​

vue
<template>
  <div>
    <h2>リアクティビティの例</h2>
    <input v-model="message" placeholder="メッセージを入力">
    <p>入力されたメッセージ: {{ message }}</p>
    
    <h3>計算プロパティ</h3>
    <p>文字数: {{ messageLength }}</p>
    <p>逆順: {{ reversedMessage }}</p>
    
    <h3>ウォッチャー</h3>
    <p>変更回数: {{ changeCount }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: '',
      changeCount: 0
    }
  },
  computed: {
    messageLength() {
      return this.message.length;
    },
    reversedMessage() {
      return this.message.split('').reverse().join('');
    }
  },
  watch: {
    message(newValue, oldValue) {
      this.changeCount++;
      console.log(`メッセージが "${oldValue}" から "${newValue}" に変更されました`);
    }
  }
}
</script>

リアクティブなオブジェクトと配列 ​

vue
<template>
  <div>
    <h2>リアクティブなオブジェクトと配列</h2>
    
    <h3>ユーザー情報</h3>
    <input v-model="user.name" placeholder="名前">
    <input v-model="user.email" placeholder="メールアドレス">
    <p>名前: {{ user.name }}</p>
    <p>メール: {{ user.email }}</p>
    
    <h3>タスクリスト</h3>
    <input v-model="newTask" @keyup.enter="addTask" placeholder="新しいタスク">
    <ul>
      <li v-for="(task, index) in tasks" :key="index">
        {{ task }}
        <button @click="removeTask(index)">削除</button>
      </li>
    </ul>
  </div>
</template>

<script>
export default {
  data() {
    return {
      user: {
        name: '',
        email: ''
      },
      tasks: [],
      newTask: ''
    }
  },
  methods: {
    addTask() {
      if (this.newTask.trim()) {
        this.tasks.push(this.newTask.trim());
        this.newTask = '';
      }
    },
    removeTask(index) {
      this.tasks.splice(index, 1);
    }
  }
}
</script>

ディレクティブ ​

Vue.jsでは、ディレクティブを使用してDOMを操作します。

基本的なディレクティブ ​

vue
<template>
  <div>
    <h2>ディレクティブの例</h2>
    
    <!-- v-if / v-else -->
    <div v-if="isVisible">
      <p>この要素は表示されています</p>
    </div>
    <div v-else>
      <p>この要素は非表示です</p>
    </div>
    <button @click="toggleVisibility">表示/非表示を切り替え</button>
    
    <!-- v-show -->
    <div v-show="isVisible">
      <p>v-showで制御された要素</p>
    </div>
    
    <!-- v-for -->
    <h3>ユーザーリスト</h3>
    <ul>
      <li v-for="user in users" :key="user.id">
        {{ user.name }} ({{ user.email }})
      </li>
    </ul>
    
    <!-- v-bind -->
    <div :class="{ active: isActive }" :style="{ color: textColor }">
      動的なクラスとスタイル
    </div>
    
    <!-- v-model -->
    <input v-model="inputValue" placeholder="入力してください">
    <p>入力値: {{ inputValue }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: true,
      isActive: true,
      textColor: 'blue',
      inputValue: '',
      users: [
        { id: 1, name: '田中太郎', email: 'tanaka@example.com' },
        { id: 2, name: '佐藤花子', email: 'sato@example.com' },
        { id: 3, name: '鈴木一郎', email: 'suzuki@example.com' }
      ]
    }
  },
  methods: {
    toggleVisibility() {
      this.isVisible = !this.isVisible;
    }
  }
}
</script>

コンポーネント ​

Vue.jsでは、コンポーネントを使用してUIを再利用可能な部品に分割できます。

基本的なコンポーネント ​

vue
<!-- UserCard.vue -->
<template>
  <div class="user-card">
    <h3>{{ user.name }}</h3>
    <p>{{ user.email }}</p>
    <button @click="editUser">編集</button>
    <button @click="deleteUser">削除</button>
  </div>
</template>

<script>
export default {
  name: 'UserCard',
  props: {
    user: {
      type: Object,
      required: true
    }
  },
  emits: ['edit', 'delete'],
  methods: {
    editUser() {
      this.$emit('edit', this.user);
    },
    deleteUser() {
      this.$emit('delete', this.user.id);
    }
  }
}
</script>

親コンポーネントでの使用 ​

vue
<!-- App.vue -->
<template>
  <div id="app">
    <h1>ユーザー管理システム</h1>
    
    <div class="user-form">
      <h2>新しいユーザーを追加</h2>
      <input v-model="newUser.name" placeholder="名前">
      <input v-model="newUser.email" placeholder="メールアドレス">
      <button @click="addUser">追加</button>
    </div>
    
    <div class="user-list">
      <h2>ユーザー一覧</h2>
      <UserCard 
        v-for="user in users" 
        :key="user.id" 
        :user="user"
        @edit="editUser"
        @delete="deleteUser"
      />
    </div>
  </div>
</template>

<script>
import UserCard from './components/UserCard.vue';

export default {
  name: 'App',
  components: {
    UserCard
  },
  data() {
    return {
      newUser: {
        name: '',
        email: ''
      },
      users: [
        { id: 1, name: '田中太郎', email: 'tanaka@example.com' },
        { id: 2, name: '佐藤花子', email: 'sato@example.com' }
      ],
      nextId: 3
    }
  },
  methods: {
    addUser() {
      if (this.newUser.name && this.newUser.email) {
        this.users.push({
          id: this.nextId++,
          name: this.newUser.name,
          email: this.newUser.email
        });
        this.newUser = { name: '', email: '' };
      }
    },
    editUser(user) {
      const newName = prompt('新しい名前を入力してください:', user.name);
      const newEmail = prompt('新しいメールアドレスを入力してください:', user.email);
      
      if (newName && newEmail) {
        user.name = newName;
        user.email = newEmail;
      }
    },
    deleteUser(userId) {
      if (confirm('このユーザーを削除しますか？')) {
        this.users = this.users.filter(user => user.id !== userId);
      }
    }
  }
}
</script>

ライフサイクルフック ​

Vue.jsでは、コンポーネントのライフサイクルの各段階で実行されるフックを提供しています。

vue
<template>
  <div>
    <h2>ライフサイクルフックの例</h2>
    <p>コンポーネントが作成されてから {{ timeElapsed }} 秒経過しました</p>
    <button @click="updateData">データを更新</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      timeElapsed: 0,
      interval: null
    }
  },
  beforeCreate() {
    console.log('beforeCreate: インスタンスが作成される前');
  },
  created() {
    console.log('created: インスタンスが作成された後');
    this.interval = setInterval(() => {
      this.timeElapsed++;
    }, 1000);
  },
  beforeMount() {
    console.log('beforeMount: DOMにマウントされる前');
  },
  mounted() {
    console.log('mounted: DOMにマウントされた後');
  },
  beforeUpdate() {
    console.log('beforeUpdate: データが更新される前');
  },
  updated() {
    console.log('updated: データが更新された後');
  },
  beforeUnmount() {
    console.log('beforeUnmount: インスタンスが破棄される前');
    if (this.interval) {
      clearInterval(this.interval);
    }
  },
  unmounted() {
    console.log('unmounted: インスタンスが破棄された後');
  },
  methods: {
    updateData() {
      this.timeElapsed = 0;
    }
  }
}
</script>

ルーティング（Vue Router） ​

Vue Routerを使用して、シングルページアプリケーション（SPA）のルーティングを実装できます。

インストールとセットアップ ​

bash
npm install vue-router@4
javascript
// router/index.js
import { createRouter, createWebHistory } from 'vue-router';
import Home from '../views/Home.vue';
import About from '../views/About.vue';
import User from '../views/User.vue';

const routes = [
  {
    path: '/',
    name: 'Home',
    component: Home
  },
  {
    path: '/about',
    name: 'About',
    component: About
  },
  {
    path: '/user/:id',
    name: 'User',
    component: User,
    props: true
  }
];

const router = createRouter({
  history: createWebHistory(),
  routes
});

export default router;

状態管理（Pinia） ​

Piniaは、Vue.jsの公式状態管理ライブラリです。

インストールとセットアップ ​

bash
npm install pinia
javascript
// main.js
import { createApp } from 'vue';
import { createPinia } from 'pinia';
import App from './App.vue';
import router from './router';

const app = createApp(App);
app.use(createPinia());
app.use(router);
app.mount('#app');

ストアの作成 ​

javascript
// stores/user.js
import { defineStore } from 'pinia';

export const useUserStore = defineStore('user', {
  state: () => ({
    users: [
      { id: 1, name: '田中太郎', email: 'tanaka@example.com' },
      { id: 2, name: '佐藤花子', email: 'sato@example.com' }
    ],
    currentUser: null
  }),
  
  getters: {
    getUserById: (state) => (id) => {
      return state.users.find(user => user.id === id);
    },
    userCount: (state) => state.users.length
  },
  
  actions: {
    addUser(user) {
      const newUser = {
        id: Date.now(),
        ...user
      };
      this.users.push(newUser);
    },
    
    updateUser(id, updates) {
      const user = this.users.find(user => user.id === id);
      if (user) {
        Object.assign(user, updates);
      }
    },
    
    deleteUser(id) {
      this.users = this.users.filter(user => user.id !== id);
    },
    
    setCurrentUser(user) {
      this.currentUser = user;
    }
  }
});

ストアの使用 ​

vue
<!-- UserManagement.vue -->
<template>
  <div>
    <h2>ユーザー管理</h2>
    
    <div class="user-form">
      <h3>新しいユーザーを追加</h3>
      <input v-model="newUser.name" placeholder="名前">
      <input v-model="newUser.email" placeholder="メールアドレス">
      <button @click="addUser">追加</button>
    </div>
    
    <div class="user-list">
      <h3>ユーザー一覧 ({{ userCount }}人)</h3>
      <div v-for="user in users" :key="user.id" class="user-item">
        <span>{{ user.name }} ({{ user.email }})</span>
        <button @click="editUser(user)">編集</button>
        <button @click="deleteUser(user.id)">削除</button>
      </div>
    </div>
    
    <div v-if="currentUser" class="current-user">
      <h3>現在選択中のユーザー</h3>
      <p>{{ currentUser.name }} ({{ currentUser.email }})</p>
    </div>
  </div>
</template>

<script>
import { useUserStore } from '../stores/user';

export default {
  name: 'UserManagement',
  setup() {
    const userStore = useUserStore();
    
    return {
      userStore
    };
  },
  data() {
    return {
      newUser: {
        name: '',
        email: ''
      }
    };
  },
  computed: {
    users() {
      return this.userStore.users;
    },
    userCount() {
      return this.userStore.userCount;
    },
    currentUser() {
      return this.userStore.currentUser;
    }
  },
  methods: {
    addUser() {
      if (this.newUser.name && this.newUser.email) {
        this.userStore.addUser(this.newUser);
        this.newUser = { name: '', email: '' };
      }
    },
    editUser(user) {
      const newName = prompt('新しい名前を入力してください:', user.name);
      const newEmail = prompt('新しいメールアドレスを入力してください:', user.email);
      
      if (newName && newEmail) {
        this.userStore.updateUser(user.id, {
          name: newName,
          email: newEmail
        });
      }
    },
    deleteUser(id) {
      if (confirm('このユーザーを削除しますか？')) {
        this.userStore.deleteUser(id);
      }
    }
  }
};
</script>

まとめ ​

Vue.jsは、親しみやすく、パフォーマンスと汎用性の高いプログレッシブJavaScriptフレームワークです。Vue.js公式サイトで提供されている豊富なドキュメントと、直感的なAPIにより、初心者でも簡単に学習できます。

Vue.jsの主な特徴 ​

  • 親しみやすい: 標準的なHTML、CSS、JavaScriptをもとに構築
  • 高パフォーマンス: 真にリアクティブなコンパイラで最適化されたレンダリングシステム
  • 多用途: ライブラリーとフル機能のフレームワークの間でスケール
  • 段階的採用: 既存のプロジェクトに部分的に導入可能

学習のポイント ​

  • 基本的な構文から始める
  • リアクティビティの概念を理解する
  • コンポーネントの作成と使用方法を学ぶ
  • ルーティングと状態管理を実践する
  • 実際のプロジェクトで使用して経験を積む

推奨される学習順序 ​

  1. 基本的な構文 - テンプレート、データバインディング
  2. ディレクティブ - v-if、v-for、v-modelなど
  3. コンポーネント - 再利用可能なUI部品の作成
  4. ライフサイクルフック - コンポーネントのライフサイクル
